# CIFAR Legacy Scripts

This folder holds older Spurious CIFAR / CifarMnist scripts that are no longer
the preferred first entrypoints.

Use these files only for:

- archaeology
- narrow diagnostics
- reproducing older notebook-style observations
- rerunning one-off ablations that are no longer on the main thesis path

The active CIFAR entrypoints remain in the parent `cifar/` directory and are
documented in `cifar/README.md`.
